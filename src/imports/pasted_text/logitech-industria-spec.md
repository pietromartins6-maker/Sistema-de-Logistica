---

# CONTEXTO DO PROJETO

O projeto NÃO é um curso online.

O projeto NÃO é uma plataforma de ensino.

O projeto NÃO é um ERP completo.

O projeto NÃO deve possuir aparência de um site institucional.

O objetivo é desenvolver um **software operacional** que simule os processos logísticos de uma empresa fictícia chamada **LogiTech Indústria**.

A empresa fabrica equipamentos eletrônicos para o setor industrial.

O usuário assume o papel de Assistente de Logística e acompanha as operações da empresa.

Todo o conteúdo deve ser apresentado através das funcionalidades do sistema.

A teoria deve aparecer apenas como apoio ao processo operacional.

O usuário deve aprender observando a operação da empresa, e não lendo grandes textos.

---

# OBJETIVO PRINCIPAL

O sistema deve demonstrar os conhecimentos adquiridos no Curso Técnico em Logística através de processos reais encontrados em uma indústria.

* indicadores;

# OBJETIVO FINAL

Desenvolver um software web moderno, elegante e altamente interativo que simule os principais processos logísticos de uma indústria fictícia. O sistema deve demonstrar, por meio de operações, indicadores, dashboards, ferramentas e simulações, a aplicação prática dos conhecimentos adquiridos no Curso Técnico em Logística. O resultado esperado não é um curso nem um ERP completo, mas um **Centro de Operações Logísticas Educacional**, com aparência de software corporativo, profundidade funcional, excelente experiência do usuário e alto valor como portfólio técnico. Esse projeto deve ser visualmente marcante, tecnicamente organizado e coerente com a realidade de um estudante do Ensino Médio Técnico, demonstrando domínio dos processos logísticos de forma prática e profissional.
